== Description ==

This plugin adds a widgets for a split sidebar and a Genesis layout option for them.

== Installation ==

1. Install the plugin
2. Activate the plugin
3. Add widgets to new sidebars
4. Select Content/Split-sidebar layout in post editor.

== Frequently Asked Questions ==

= Is this responsive? =

It behaves as a single sidebar.

= Can I use this with Genesis Simple Sidebars? =
Yes, it should work with GSS.
